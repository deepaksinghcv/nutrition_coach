02 — The ExperimentHypothesis
Same model.
Two datasets.
Which labels win?
Fine-tune Qwen3-VL-4B twice with identical QLoRA settings — once on Food101 (synthetic class-average nutrition) and once on Nutrition5k (real lab-measured values) — then test both on the same unseen plates.
Nutrition Coach04
02 — The ExperimentData sources
The contrast that drives everything
Food101
Synthetic labels
101 food classes, but no real nutrition. Each class is mapped to one USDA-style average tuple — the same numbers for every photo of that class.
Nutrition5k
Measured labels
Real lab-measured calories & macros per dish (Google). Each image carries its actual portion's values — variance and all.

347 training images each · clean 28-dish Nutrition5k holdout for scoring.

04 — The FindingSurprise
Food101 fine-tuning was

Training on fabricated class-average labels didn't just fail to help — it actively degraded accuracy below the un-tuned base model on all four nutrients.
Nutrition Coach10
04 — The FindingMechanism
Why it backfired

Confident,
but wrong.

Forced into 101 buckets, the model collapses different dishes into a few classes — then emits each class's fixed average. It learns a confident mapping that ignores the actual plate.

Observed failure
"grilled salmon" ×8
On the holdout, Food101 labelled 8 unrelated dishes as grilled salmon — spanning 21 to 1116 kcal — and assigned them all the same calorie average.

05 — EngineeringWar stories
Things that broke (and the fixes)
  • 01MPS too slow for trainingBackward pass through the vision encoder stalled for hours on Apple Silicon → moved training to a Kaggle T4.
  • 02bf16 GradScaler crash on T4Mixed-precision scaler couldn't unscale bf16 grads → disabled AMP; 4-bit base still computes in fp16 internally.
  • 03Corrupted 3.7GB save recoveredA wrong save path bundled the quantized base → wrote a safetensors tool to extract the clean 33MB LoRA adapter.

06 — TakeawaysWhat it means
The lesson
  • →Label quality beats model tweaks.Real measured data cut calorie error ~34% vs. base; synthetic labels made it worse.
  • →A pretty loss curve can lie.Food101's 0.04 loss was memorization; Nutrition5k's 0.41 plateau was the honest signal.
  • →Next: scale the real data.Only 347 of 4,768 Nutrition5k dishes were used — more should push error lower still.
